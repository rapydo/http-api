# RAPyDo REST APIs

REST APIs integrated with the RAPyDo framework

Based on:

- Flask
- flask-apispec with apispec
- marshmallow
- webargs
- Flask-Caching
- PyJWT
- passlib
- pyotp With segno
- Flask-SQLAlchemy
- Neomodel
- PyMySQL
- Redis-py
- pika with amqp
- celery with celery-redbeat
- gunicorn with gevent
